

# Tic-Tac-Toe
Tic-Tac-Toe is a game built using Java language. it has 3 modes: single mode, 2 players and play through the network.


### Prerequisites


```
JVM
mariaDB
```

## Deployment


```
import database schema 'tic_tac_toe.sql' into your DBMS
```


```
start playing using the jar file
```
